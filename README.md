# TuboOvarianCancer_AoUAnalysis

> Analysis of All of Us Controlled Tier Dataset v8 pertaining to our manuscript on oncologic histories in tubo-ovarian cancer

Script files (in chronological order):

* ```1_prep_meta_features.py``` : Prepare metadata features for cases and controls to use downstream
* ```2_prep_age_features.py``` : Prepare age features for cases and controls to use downstream
* ```3_prep_clean_merged_dataset.py``` : Merge metadata and age files to generate a clean file for use downstream
* ```4_get_age_race_eth_preg_smok.py``` : Pull case and control counts for age, race, ethnicity, pregnancy status, and smoking status
* ```5_get_cases_pfh.py``` : Get case Personal/Family Health History for arthritis, cancers, and obesity 
* ```6_get_cases_pfh.py``` : Get control Personal/Family Health History for arthritis, cancers, and obesity
